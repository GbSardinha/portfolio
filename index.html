Talk'n'Walk APP

AngularJS

The Web APP that was developed during the EticIDEIA contest.
The APP is only developed for mobile, so to be able to see it’s workflow please use your browser Responsive Design Mode. For Windows press ctrl + shift + m. For Mac right click + inspect and then choose Responsive Design Mode on the top right corner.
In this project we had to find a solution for one of the ONU’s challenges, in which we chose to tackle the mental health issues and taboos.
The APP facilitates the contact between therapists and patients in an private environment and is intended has a springboard to help the patients get proper one-to-one therapy.
